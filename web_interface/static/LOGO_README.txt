IMPORTANT: Logo Placement
=========================

Please place the JLBMaritime logo file (logo.png) in this directory:
/web_interface/static/logo.png

The logo should be the JLB Maritime branding image with the blue circular 
radar/target icon and "JLB MARITIME" text.

This logo will be displayed in:
- Login page
- Main dashboard header
- All web interface pages

If the logo file is missing, the web interface will still function but 
may show a broken image icon where the logo should appear.
